Zero Trust Principles

Never Trust, Always Verify

Every request is authenticated and authorized via Keycloak OIDC tokens.

Short-Lived Tokens

Access tokens expire in 5 minutes. Sessions max out at 1 hour.

Secrets Never Hardcoded

All secrets fetched at runtime from OpenBao — never in code or env vars.

Least Privilege Access

Role-based access control with granular policies for each service.

Continuous Monitoring

All identity events shipped to ELK Stack for real-time threat detection.

TLS Everywhere

Nginx enforces HTTPS with HSTS headers on all traffic.
