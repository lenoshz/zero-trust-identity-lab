# Screenshots

This directory is a placeholder for screenshots of the running lab.

After deploying the lab with `./scripts/bootstrap.sh`, take screenshots of:

1. **Landing Page** — Flask app showing system status (`https://localhost/app`)
2. **Keycloak Login** — OIDC login page (`https://localhost/auth`)
3. **Dashboard** — Authenticated user dashboard with role info
4. **Secrets Page** — OpenBao secrets with masked values
5. **Kibana SIEM** — Identity threat dashboard (`https://localhost/kibana`)
6. **Health Check** — Terminal output of `./scripts/healthcheck.sh`

## Naming Convention

Save screenshots as:
- `01-landing-page.png`
- `02-keycloak-login.png`
- `03-dashboard.png`
- `04-secrets-page.png`
- `05-kibana-dashboard.png`
- `06-health-check.png`
